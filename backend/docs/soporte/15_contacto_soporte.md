## Antes de Contactar Soporte

### Intentá estas soluciones primero:

1. **Recargá la página**
   - Presioná F5 o Ctrl+R
   - O hacé clic en el botón de recargar del navegador

2. **Hard refresh**
   - Presioná Ctrl+Shift+R
   - Esto limpia la caché del navegador

3. **Cerrá sesión y volvé a entrar**
   - Hacé clic en "Cerrar sesión"
   - Volvé a loguearte

4. **Probá otro navegador**
   - Si usás Chrome, probá Firefox o Edge
   - Verificá que el navegador esté actualizado

5. **Verificá tu conexión a internet**
   - ¿Podés abrir otras páginas web?

## Cómo Reportar un Problema

### Información necesaria:

Cuando escribas al soporte, incluí:

1. **¿Qué intentabas hacer?**
   - Ejemplo: "Estaba registrando un ingreso"

2. **¿Qué pasó?**
   - Ejemplo: "Apareció un mensaje de error"

3. **¿Qué mensaje de error viste?**
   - Copiá el texto exacto del error
   - O sacá una captura de pantalla

4. **¿En qué pantalla estabas?**
   - Ejemplo: "En el Dashboard", "En el modal de Gasto"

5. **¿Qué navegador usás?**
   - Ejemplo: "Chrome 120", "Firefox", "Safari"

6. **¿Es la primera vez que pasa?**
   - ¿Funcionaba antes? ¿Cuándo dejó de funcionar?

### Ejemplo de reporte bien hecho:

```
Asunto: Error al guardar ingreso

Hola Bruno,

Estaba intentando registrar un ingreso y cuando hago clic en "Guardar" 
aparece el mensaje "Error al guardar operación" y no se guarda.

- Pantalla: Modal de Registrar Ingreso
- Navegador: Google Chrome
- ¿Primera vez?: Sí, antes funcionaba bien
- Datos que intenté cargar:
  - Cliente: Empresa ABC
  - Área: Jurídica
  - Monto: 15000 UYU

¿Me podrías ayudar?

Gracias,
María
```

## Tipos de Problemas

### Problemas Urgentes (contactar inmediatamente)
- ❗ No puedo entrar al sistema (nadie puede)
- ❗ El sistema está caído
- ❗ Se perdieron datos importantes
- ❗ Error de seguridad

### Problemas Normales (pueden esperar)
- ⚠️ Un botón no funciona
- ⚠️ Los números no cuadran
- ⚠️ No entiendo cómo hacer algo
- ⚠️ Sugerencia de mejora

### No es un problema técnico
- ℹ️ Olvidé mi contraseña → Pedí reset a un socio
- ℹ️ No sé cómo cargar algo → Revisá esta documentación
- ℹ️ Quiero aprender a usar el CFO AI → Revisá la sección 09

## Preguntas Frecuentes de Soporte

### "El sistema está lento"
**Posibles causas:**
- Tu conexión a internet
- Muchos usuarios conectados
- Problema del servidor

**Solución:** Esperá unos minutos y recargá

### "No me deja entrar"
**Verificá:**
- ¿Escribiste bien el email?
- ¿Recordás la contraseña?
- ¿Tu cuenta está activa?

**Solución:** Pedí reset de contraseña a un socio

### "Perdí una operación que cargué"
**Verificá:**
- ¿Apareció el mensaje de éxito?
- ¿Están bien los filtros de fecha?
- ¿Otro usuario la eliminó?

**Solución:** Revisá el panel de Operaciones con filtros amplios

### "Los números del Dashboard no coinciden con mis cálculos"
**Verificá:**
- ¿Los filtros están correctos?
- ¿Estás viendo en UYU o USD?
- ¿Incluiste todas las operaciones?

**Solución:** Limpiá filtros y verificá moneda
